# 🇩🇪 German A1 Learning Website

An interactive web application designed to help beginners prepare for the German A1 language exam. The website provides essential grammar topics, vocabulary, practice exercises, and learning resources in a simple and user-friendly interface.

## 🌐 Live Demo

🔗 (https://gmoni081007-hue.github.io/germenA1/)

---

## ✨ Features

- 📚 German A1 Grammar Notes
- 📝 Practice Questions
- 📖 Vocabulary Lists
- 💡 Easy-to-understand Explanations
- 📱 Responsive Design (Mobile & Desktop)
- ⚡ Fast and Lightweight Website

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

### Open the Project

Simply open `index.html` in your browser.

Or use VS Code Live Server:

1. Open the project in VS Code.
2. Install the **Live Server** extension.
3. Right-click `index.html`.
4. Click **Open with Live Server**.
