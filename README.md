# 🇩🇪 German A1 Learning Website

An interactive web application designed to help beginners prepare for the German A1 language exam. The website provides essential grammar topics, vocabulary, practice exercises, and learning resources in a simple and user-friendly interface.

## 🌐 Live Demo

🔗 (https://gmoni081007-hue.github.io/germanA1/)

---

## ✨ Features

- 📚 German A1 Grammar Notes
- 📝 Practice Questions
- 📖 Vocabulary Lists
- 💡 Easy-to-understand Explanations
- 📱 Responsive Design (Mobile & Desktop)
- ⚡ Fast and Lightweight Website
